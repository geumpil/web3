함수는 function, 기능 이라는 뜻을 가지고 있는데
지금 우리 단계에서는 그룹으로 묶는 느낌으로 생각할 것

기본적인 문법은

function 함수이름() {
    실행될 코드;
}

함수이름(); 을 실행하면
내부에 있는 코드들을 가져와서 실행 시켜준다.

function 함수이름(매개변수) {
    실행될 코드;
    이 안쪽에서 매개변수를 이용할 수 있다.
}

매개변수는 하나만 들어가는건 아니다.
필요한만큼 사용 가능하다.

여러개 사용할 때는

function 함수이름(매개변수1, 매개변수2, 매개변수3) {
    실행될 코드;
}
 
같은 형태로 사용한다.

반환값이 있는 함수 : alert()
반환값이 없는 함수 : prompt(), confirm()
